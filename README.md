# Customer Churn Prediction

## 📊 Project Overview
Built a Machine Learning model to predict customer 
churn for a Telecom company using Python and Sklearn.

## 🛠️ Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Scikit-learn (Logistic Regression, Random Forest, XGBoost)
- Google Colab
- GitHub

## 📈 Models Built
| Model | Accuracy |
|-------|----------|
| Logistic Regression | 81.6% |
| Random Forest | 79.35% |
| XGBoost | 78.14% |

## 🔍 Steps Followed
1. Data Loading and Exploration
2. Data Cleaning and Preprocessing
3. Feature Engineering
4. Model Building
5. Model Evaluation
6. Model Comparison

## 💡 Key Insights
- Customers with month-to-month contracts churn more
- Higher monthly charges leads to more churn
- Customers with longer tenure are less likely to churn
